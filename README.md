# README

1. 가상환경을 만들고, 적용시켜 주세요

```
$ python -m venv venv
$ source venv/Scripts/activate
```

2. requirements.txt를 설치해주세요

```
$ pip install -r requirements.txt
```

3. 실행시켜 주세요!!!

```
$ python manage.py runserver
```

4. 처음 페이지를 로딩했을 때, 깨져보인다면 새로고침 한번 꼬옥 눌러주세요!!😊
